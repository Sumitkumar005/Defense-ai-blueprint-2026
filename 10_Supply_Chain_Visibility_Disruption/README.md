# Supply Chain Visibility & Disruption Prediction

## Overview
AI-powered supply chain control tower that provides end-to-end visibility, predicts disruptions, and recommends mitigation strategies.

## Key Features
1. Multi-Tier Visibility
2. Disruption Prediction
3. Alternative Source Finder
4. In-Transit Tracking
5. Risk Scoring
6. Scenario Planning
7. Cyber Threat Integration
8. Inventory Optimization
9. Supplier Performance
10. Compliance Monitoring

## Tech Stack
- **Backend**: Python FastAPI + Node.js
- **Graph Database**: Neo4j
- **Time-Series**: InfluxDB
- **ML**: Scikit-learn, Prophet
- **NLP**: News/report processing
- **Visualization**: D3.js
- **Message Queue**: RabbitMQ
