# Peer Support Network & Wellness Check System

## Overview
Structured peer support platform that facilitates regular wellness check-ins, strengthens unit cohesion, and flags concerning patterns to leadership.

## Key Features
1. Battle Buddy Matching (AI-based)
2. Guided Check-In Prompts
3. Mood Tracking
4. Confidential Concerns Reporting
5. Resource Library
6. Unit Cohesion Metrics
7. Crisis Escalation
8. Support Group Coordination

## Tech Stack
- **Mobile**: React Native
- **Backend**: Node.js with Express
- **Database**: MongoDB + Neo4j (graph)
- **Encryption**: Signal Protocol for E2E
- **ML**: BERT for sentiment analysis
- **Real-Time**: Socket.io
